So in this project, i want to create a new webapp that will help me with writing. Writing blogs/articles, letters, reports and docs.

The way I want it to be is, there will be projects, and inside each project there will dirs/files/sub-dirs. All of them will be Markdown files. These are resources. Then i will read those files, and make notes/highlight them. I will never edit them.

And what i want is an AI system that will read these resources, come up with interesting lenses for the doc like a financial expert, or a real-estate expert or a political expert or a software engineering expert, depending on the doc, and these ai experts will have their own notes, which i can review and discard if i want to or add to my notes, with additional changes if needed.

Me and the AI will do this round of reading together on the set of docs, and I will be the one creating a dump of notes with snippets and highlights and my thoughts and ai's thoughts.

And then there should be a button, that will use these notes, to create a report for me.

Once that report is created, the mode shifts to editor for me. I can go in edit paragraph by manually typing. And here paragraph is a unit block.

Per block i want 2 options: 1. Change of tone which will tell ai to generate 5 samples based on different tone, based ont he context of the report.
2. I want the AI to start critiquing my argument and challenging me, so that i can better formulate my argument.
